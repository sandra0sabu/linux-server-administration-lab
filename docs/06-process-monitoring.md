# Module 06 - Process & Performance Monitoring

## Objective

Learn to monitor Linux system performance.

---

## Commands Used

```bash
top
free -h
df -h
uptime
ps -ef
vmstat
```

---

## What I Learned

- Monitor CPU utilization
- Monitor memory usage
- Monitor disk usage
- View running processes
- Check system uptime

---

## Interview Questions

### What does top command do?

Displays running processes and system resource usage.

---

### Difference between free and df?

free → Memory

df → Disk Space
